# Security Policy

## Supported Versions

Security fixes are handled on the latest released version.

## Reporting A Vulnerability

Please do not open a public issue for security-sensitive reports.

Report privately through GitHub Security Advisories for this repository when available, or contact the maintainer directly from the GitHub profile.

Include:

- affected version or commit
- reproduction steps
- expected impact
- whether the issue affects generated context packs, repository scanning, CI checks, or MCP usage
